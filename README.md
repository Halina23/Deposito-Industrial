# Sistema de Controle de Depósito Industrial (MySQL)
Projeto de banco de dados relacional desenvolvido para gerenciar estoque, movimentações, produtos, localizações e regras de negócio de um depósito industrial.
Este projeto foi criado com foco em modelagem, integridade dos dados, automação com triggers, consultas analíticas e geração de relatórios.

Projeto desenvolvido para fins de aprendizado, portfólio e validação de boas práticas em SQL/MySQL.
